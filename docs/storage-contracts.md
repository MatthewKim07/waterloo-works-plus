# `chrome.storage.local` keys

| Key | Module | Purpose |
|-----|--------|---------|
| `wwpSettings` | `storage.js` | Enabled, flags, resume text, prefs |
| `wwpJobAnalysisCache` | `storage.js` | Cached job parses |
| `wwpProfiles` | `profile-store.js` | Named resume profiles |
| `wwpApplications` | `application-store.js` | Tracker rows |
| `wwpEventLog` | `event-log.js` | Recent extension events (trimmed list) |
| `wwpDebug` | `logger.js` | Verbose logging toggle |
